SHIELDRESPONSE: SETUP & LAUNCH GUIDE
Welcome to the ShieldResponse Simulation. Please follow these steps to get the project running on your machine.

🛠️ PART 1: INSTALL THE CORE TOOLS (Do this only once)
You need these two programs installed for the simulation to run.

PYTHON: Download Here "https://www.python.org/downloads/"

⚠️ CRITICAL: When the Python installer starts, you MUST check the box at the very bottom that says "Add Python.exe to PATH" before clicking Install.

NODE.JS: Download Here "https://nodejs.org/" (Choose the "LTS" version).

📥 PART 2: INSTALL THE GAME (Do this only once)
Now that your computer has the right tools, we need to install the game's specific files.

Open the main ShieldResponse folder.

Double-click First_Time_Setup.bat.

A black window will open and download all the necessary files. This may take a few minutes.

When it says "INSTALLATION COMPLETE!", press any key to close the window.

🚀 PART 3: PLAY THE GAME (Do this every time)
Double-click Launch_ShieldResponse.bat.

Two terminal windows will open automatically to run the servers. Do not close them!

Wait about 5 seconds, then hold CTRL and click the http://localhost:5173 link that appears in the Frontend window to play.
